# hangman-bash
Simple hangman game in bash by jc.

### Versions:
### Hangman v1.0
Located in hangman_v1/ <br>
The first version provides simple functionality to start, play and end game.

### Hangman v2.0
Located in hangman_v2/ <br>
Extended version with tracking of player name and player statistics - games played and wins, from which a leaderboard is created including percentage and rating.
